## نصب لوگو شرکت
فایل لوگو را با نام `logo.png` در کنار `app.py` قرار دهید. (اختیاری)

## اجرای محلی
```bash
pip install -r requirements.txt
streamlit run app.py
```

## دیپلوی روی لیارا
```bash
liara deploy
```
فضای ذخیره‌سازی `visitor-data` روی مسیر `/app/data` مانت می‌شود.
داده‌ها در این دیسک ذخیره شده و بین ری‌استارت‌ها حفظ می‌شوند.
